Changelog
=========

All notable changes to SITS-Former will be documented in this file.

Version 0.1.0 (2025-12-01)
---------------------------

**Added**

* Initial release of SITS-Former
* Transformer-based architecture for satellite image time series
* Support for multi-class land cover classification
* PyTorch implementation with GPU acceleration
* Command-line interface for training, evaluation, and inference
* Comprehensive configuration system
* Integration with Weights & Biases for experiment tracking
* TensorBoard support for training visualization

**Features**

* Multi-head attention mechanism for temporal sequence modeling
* Configurable model architecture (depth, width, attention heads)
* Data augmentation techniques for satellite imagery
* Multiple evaluation metrics (accuracy, F1-score, IoU)
* Checkpoint saving and loading
* Early stopping with patience
* Learning rate scheduling

**Documentation**

* Complete API documentation
* Installation and quick start guides
* Tutorial notebooks for common use cases
* Contributing guidelines
