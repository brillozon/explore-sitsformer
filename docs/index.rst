SITS-Former: Satellite Image Time Series Transformer
=====================================================

SITS-Former is a PyTorch implementation of a transformer-based model for satellite image time series analysis and classification. It's designed for processing and analyzing temporal sequences of satellite imagery for various remote sensing applications.
